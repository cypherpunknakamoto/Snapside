## What is SnapSide?

SnapSide is a productivity app for macOS that revolutionizes how you communicate with AI assistants. Instead of typing long descriptions of what you're looking at, just capture your screen and let AI see exactly what you're working on.

### The Problem
Ever find yourself typing out a long description of what's on your screen to ChatGPT, Claude, or Perplexity? Describing error messages, code snippets, or design layouts takes time and often loses important context.

### The Solution
SnapSide adds a floating button to your screen. One click captures your current window and copies it to your clipboard‚Äîready to paste directly into any AI conversation. Show, don't tell.

## Getting Started

### First Launch
1. Open SnapSide from Applications
2. Grant Screen Recording permission when prompted
   - Go to System Settings > Privacy & Security > Screen Recording
   - Enable SnapSide
3. The floating button appears in the top-right corner
4. You're ready to go!

### Common Issues

**Q: The button doesn't appear**
A: Make sure Screen Recording permission is enabled in System Settings > Privacy & Security

**Q: Screenshots are blank**
A: Some apps block screen recording. Try capturing a different window.

**Q: Keyboard shortcut not working**
A: Check if another app is using ‚åò‚áß6. You can customize shortcuts in Settings.

**Q: How do I uninstall?**
A: Drag SnapSide from Applications to Trash, then remove permissions from System Settings.
